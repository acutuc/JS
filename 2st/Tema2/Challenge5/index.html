Challenge V. Inversión de arrays.

Realiza un programa al que se le introduzca un texto y devuelva el mismo texto pero con las letras invertidas. Por ejemplo la entrada:
Buenos días hoy es lunes
Daría la salida:
soneuB saíd yoh se senul
***************************************** AYUDA *************************************************
Un string en JS es como un array.
Trucos:
Un string en JS es como un array. Puedes acceder a la posición 5 haciendo cadena[4]
también puedes utilizar el length. for (let i = cadena.length - 1; i >= 0; i--) {
Funciones que puedes utilizar:
La función split() devuelve un array de elementos. Si le pasas un delimitador le servirá para distinguir cada elemento.
let arrcadena = cadena.split("|");
La función replace() puede ser también útil. cadena.replace(" ","|")
Puedes encadenar funciones. cadena = cadena.replace(" ","|").replace(".","|").replace(",","|").replace(";","|");
