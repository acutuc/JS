JavaScript Arrays. 2 arrays ordenados.

Tenemos dos arrays arr1 y arr2 y queremos insertarlos de forma ordenada en un tercer array arr3.

Importante:
*Hay que insertarlo de forma ordenada en un tercer array.
*Solo puedes utilizar el método de arrays push() y length().
*No puedes modificar los arrays arr1 ni arr2.
**No puedes crear copias de los arrays
*No puedes crear más variables que las que se te dan.
*Muestra el resultado (arr3) en el campo demo.
*Tu programa deberá funcionar para dos arrays ordenados de cualquier tamaño.
*Los números se pueden repetir siempre que estén ordenados.
